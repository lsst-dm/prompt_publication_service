# prompt_publication_service

Service responsible for replicating datasets and metadata between
[Butler](https://github.com/lsst/daf_butler) repositories and managing the life
cycle of these datasets for
[Rubin Observatory's](https://rubinobservatory.org/) Prompt Data Products.

[DMTN-330](https://dmtn-330.lsst.io/) gives an overview of the rationale and
design for this service.
